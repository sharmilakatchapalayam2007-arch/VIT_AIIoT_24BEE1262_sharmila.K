                                                  Smart Load Monitoring System


Project Title:
Smart Load Monitoring System Using HX711 Load Cell Sensor


Description:
This project monitors the weight of materials using an HX711 load cell sensor and Arduino Uno.
The system continuously measures the weight and displays alerts when the load is below or above predefined threshold values.

Software Used:
Wokwi Simulator
Arduino IDE
GitHub
Microsoft Word

Features:
    Continuous weight monitoring
    Low Material Alert
    Normal Load Indication
    Overload Alert
    IoT-ready design for future cloud integration
